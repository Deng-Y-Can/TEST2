# Candy工具导航

## 项目简介

Candy工具导航是一个实用的在线工具集合平台，提供多种常用网页工具，包括图片处理、文件转换、代码开发、数据管理等功能。所有工具均可在浏览器本地运行，保护您的隐私安全。

## 主要功能

### 文件处理工具
- **JSON工具** - JSON格式化、验证、差异对比、结构分析、数据生成、加密压缩等
- **PDF工具** - PDF转TXT、HTML、Word、Markdown等多种格式
- **XML转JSON工具** - XML与JSON格式互相转换，支持多种数据格式
- **加密解密工具** - 支持AES、DES、RSA、Base64、MD5、SHA-256等多种加密算法
- **文件扫描工具** - 文档扫描模拟器，支持图片处理和PDF导出
- **文件格式转换工具** - Word、Excel、PDF、TXT等多种格式互转
- **文件预览工具** - 全能文件查看器，支持多种文件类型

### 图片工具
- **二维码生成器** - 生成各种类型的二维码和条形码
- **图片排版工具** - 对图片进行在线排版
- **图片自由排版** - 灵活的图片组合排版工具
- **图片格式转换** - 支持JPG、PNG、GIF、WebP等多种格式
- **图片魔法工坊** - 为图片添加各种特效变换

### 代码开发工具
- **JavaScript工具** - 在线编辑和运行JavaScript代码
- **JavaScript转TypeScript** - 将JS代码转换为TS代码
- **Python工具** - Python代码在线编辑和运行
- **C#代码编辑器** - C#代码在线编辑工具
- **代码格式化工具** - 多种编程语言代码格式美化
- **网页编辑器** - 在线编辑HTML、CSS、JavaScript
- **正则表达式工具** - 生成和测试正则表达式

### 数据库管理
- **数据库管理** - 创建数据库/表、CRUD操作、SQL执行

### 实用小工具
- **世界时钟** - 查看世界各地的当前时间
- **计算器** - 简单实用的计算器
- **单位转换器** - 长度、重量、温度等多种单位转换
- **书签导航工具** - 将书签转换为网页
- **图表生成** - 生成折线图、柱状图、饼图等
- **元素周期表** - 学习化学元素知识

### 其他工具
- **文本比较工具** - 对比两个文件的内容差异
- **画笔工具** - 在线绘画工具
- **颜色选择器** - 精确选择各种颜色
- **视频工具** - 各种视频格式转换
- **网页工具** - 网站导航集合
- **书籍导航** - 浏览精选书籍资源

## 技术特点

- **纯前端实现** - 无需服务器，所有工具在浏览器本地运行
- **响应式设计** - 适配各种设备屏幕尺寸
- **现代化UI** - 使用Tailwind CSS构建美观的界面
- **无需安装** - 直接在浏览器中打开即可使用
- **隐私保护** - 文件处理在本地完成，不会上传到服务器

## 使用说明

1. 直接打开 `index.html` 文件即可进入工具导航页面
2. 根据分类选择需要使用的工具
3. 点击相应的工具卡片即可进入使用
